# SteveEasyJobs - FYP
This example demonstrates the use of fyp-API 

### Installing
```
npm install
```
### Running
```
npm start
```
### Testing

### Markers Input:
```
http://localhost:8099/form 
```

### GET Markers:
```
http://localhost:8099/api/markers
```

### GET Markers with video ID:
```
http://localhost:8099/api/markers/(vIDhere)
```

### GET Seamarks:
```
http://localhost:8099/api/seamarks
```
